# ノーランチャート：日本政党・海外比較・政治理論の位置づけ

[Nolan Chart](https://nakadanobuhiro.github.io/nolan-chart-jp/nolan-chart-jp.html)

ジョナサン・ハイトの**道徳基盤理論**（6つの道徳基盤）をもとに、日本の主要政党・海外政党・政治理論をノーランチャート上に可視化したインタラクティブHTMLアプリです。

---

## 概要

**ノーランチャート**は、政治的立場を2軸で表現するフレームワークです。

| 軸 | 左端 | 右端 |
|----|------|------|
| X軸（横） | 大きな政府（経済介入） | 小さな政府（経済的自由） |
| Y軸（縦） | 権威主義（社会統制重視） | 自由主義（個人の自由重視） |

各政党・概念の座標は、ハイトの6道徳基盤スコアから数式で自動算出します（一部手動調整あり）。

## 使い方

1. `nolan-chart-jp.html` をブラウザで開く（ローカルファイルで動作、サーバー不要）
2. **マウスホイール**：チャートをズームイン／ズームアウト（カーソル位置を中心に拡縮）
3. **ドラッグ**：ズーム時にチャートをパン（移動）
4. **＋／－ボタン**：ズームイン／アウト　**↺ リセット**：表示を初期状態に戻す
5. **チェックボックス**：個別の表示／非表示を切替
6. **「すべてON/OFF」ボタン**：セクション全体を一括切替
7. **セクションラベルクリック**：セクションを折りたたみ／展開
8. **マウスオーバー**（チャート上のアイコン・左パネルのカード）：名称と説明を表示
9. **カードクリック**：ハイトの6道徳基盤スコアをバーチャートで展開 ＆ チャート上でハイライト

## ハイトの6道徳基盤（hxy関数）

| 変数 | 道徳基盤 | 説明 |
|------|----------|------|
| h1 | ケア/危害 | 他者への配慮・福祉重視 |
| h2 | 公平/欺き | 公正・平等への関心 |
| h3 | 忠誠/裏切り | 集団・国家への忠誠心 |
| h4 | 権威/転覆 | 階層・権威の尊重 |
| h5 | 純粋/堕落 | 文化的純粋性・伝統 |
| h6 | 自由/抑圧 | 個人的自由・反抑圧 |

### 座標変換式

```
X（経済的自由） = clamp( (h6 - h1 - 0.5×h2 - 0.3×h3 - 0.1×h4 + 120) / 150 )
Y（個人的自由） = clamp( (h6 - h4 - 0.5×h5 - 0.3×h3 + 180) / 280 )
```

## 収録データ

### 日本の政党（12党）

自由民主党、公明党、立憲民主党、日本維新の会、国民民主党、日本共産党、れいわ新選組、社会民主党、参政党、日本保守党、チームみらい、中道改革連合

### 海外の主要政党・国家（7件）

米国共和党、米国民主党、中国共産党、シーランド公国、フリーステート・プロジェクト、リベルランド、エストニア

### 政治理論・国家形態（6概念）

無政府主義（アナキズム）、無政府資本主義、夜警国家（最小国家）、福祉国家（社会民主主義）、権威主義的資本主義、全体主義

---

## 注意事項

- スコアは各政党・概念の言説を基に**概念的に推定した相対値（0〜100）**です。
- 実際の政策・選挙公約とは異なる場合があります。比較のための参照点としてお使いください。
- 一部の概念（無政府主義、全体主義）は数式による算出が困難なため、X座標を手動調整しています。
